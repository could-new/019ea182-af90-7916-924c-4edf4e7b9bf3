# ATS-Friendly Resume App

A clean, semantic, and simple Flutter application designed to showcase a resume in an ATS-friendly (Applicant Tracking System) format. The layout is optimized to be easily readable, using straightforward typography and structure that minimizes complex styling issues.

## Features
- **Clean Structure:** Uses simple headers and lists to present experience and skills.
- **Cross-Platform:** Runs seamlessly on Mobile, Web, Desktop, and Tablet.
- **Responsive:** Adjusts to any screen size, ensuring readability on all devices.
- **Selectable Text:** Uses `SelectableText` to ensure that content can be easily copied and parsed by ATS tools or recruiters.

## How to Run
1. Ensure Flutter is installed.
2. Run `flutter pub get` to fetch dependencies.
3. Start the application:
   - For web: `flutter run -d chrome`
   - For desktop: `flutter run -d macos` (or `windows`, `linux`)
   - For mobile: `flutter run` on an emulator or connected device.
